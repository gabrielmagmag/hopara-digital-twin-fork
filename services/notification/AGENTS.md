# Notification Service

Node.js + TypeScript. Serviço de notificações (porta 8085).

## Comandos principais

```bash
yarn start              # Desenvolvimento (nodemon)
yarn build              # Compila TypeScript para out/
yarn test               # Testes
yarn lint               # Lint
```

## Estrutura

```
src/            # Código fonte TypeScript
out/            # Compilado — não editar
```

## Convenções

- TypeScript strict
- Diretório `out/` gerado pelo build — não editar
